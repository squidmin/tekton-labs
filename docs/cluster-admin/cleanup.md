# Cleanup


<details>
<summary>Delete the default "minikube" cluster</summary>

To delete the default `minikube` cluster:

```shell
minikube delete
```

</details>


<details>
<summary>Delete all minikube clusters</summary>

```shell
minikube delete --all
```
